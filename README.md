# 🚗 AI Vehicle Damage Detection + Insurance Automation

## 📌 Overview
An AI-powered system that detects vehicle damage using Computer Vision and automates insurance workflows using n8n.

## 🛠️ Tech Stack
- YOLOv8 (DL/CV)
- FastAPI (Backend)
- React + Vite (Frontend)
- n8n (Automation)

## 🚀 Features
- Damage detection (scratch, dent, crack)
- Severity analysis
- Automated claim workflow
- Dashboard UI (WIP)
